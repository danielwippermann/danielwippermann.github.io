<virtual> getProtocolVersion() → {number}

Returns the protocol version of this Header instance as a 8-bit number. The high nibble is used for the major version, the low nibble for the minor version. For example: a header with protocol version 2.0 would return `0x20`. Must be implemented by sub-class.
