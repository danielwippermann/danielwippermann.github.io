Class: ConfigurationOptimizer

ConfigurationOptimizer

The ConfigurationOptimizer can be used to find the optimal set of configuration values for a given VBus device.

new ConfigurationOptimizer(deviceAddress)

Creates a new ConfigurationOptimizer for the given VBus device address.

Parameters:
Name Type Description
deviceAddress number

VBus device address of the device to configure.
