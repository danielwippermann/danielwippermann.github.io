Class: DLxJsonConverter

DLxJsonConverter

The DLxJsonConverter class takes HeaderSet instances, converts them into JSON representation and then publishes that on the readable stream side of itself. The format of the JSON data is similar to the one that is generated by the Dataloggers DL2 and DL3.

It does not support parsing JSON content back into HeaderSet instances (the writable stream side).
