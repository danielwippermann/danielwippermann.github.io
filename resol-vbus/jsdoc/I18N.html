t(key, args) → {string}

Get a translation for the given key. If more than one argument is given, the translation is then used as a format string for the I18N#sprintf method.

Parameters:
Name Type Argument Description
key string

Key for the translation
args mixed <repeatable>

Arguments to format
Source:
  • i18n.js
Returns:

Formatted string

Type
string
Example
var i18n = new I18N('de');

// outputs: Unbekanntes Gerät (0x%1$04X)
console.log(i18n.t('specification.unknownDevice'));

// outputs: Unbekanntes Gerät (0x7E11)
console.log(i18n.t('specification.unknownDevice', 0x7e11));
