Class: TextConverter

TextConverter

The TextConverter class takes header sets, converts them into text representation and then publishes that on the readable stream side of itself.

It does not support parsing text content back into header sets (the writable stream side).

new TextConverter()

Create a new TextConverter instance given the set of options.
